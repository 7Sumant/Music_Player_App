MusicApp 🎶
MusicApp is a Music Streaming Application developed using the Django Framework. This project was created as a mini project for my 6th semester and is built entirely by me. The application offers a range of features designed to enhance the user experience of streaming music.

Features
🌟 Best UI/UX Design: Enjoy a sleek and intuitive interface.
🎵 Customizable User Playlist: Create and manage your personal playlists.
📱 Interacting & User-Friendly Music Player:
Play/Pause
Repeat
Shuffle
Download music freely with credit
🎤 Popular Artist Playlist: Discover and listen to popular artists.
🔍 Music Recommendation System: Get recommendations based on your interaction.
🔐 User Authentication: Securely register, login, and logout.
📊 User Dashboard: Manage your songs and queries efficiently.
🔎 Search Bar: Easily find your favorite tracks.
⚙️ Django Admin Panel: Admin panel to manage users and songs.

bash
Copy code

Create a superuser:

bash
Copy code

Start the development server:

bash
Copy code
python manage.py runserver
Access the application:
Open your browser and go to http://127.0.0.1:8000

Usage
User Registration: Register an account to access all features.
Music Player: Use the player to play, pause, shuffle, and repeat songs.
Create Playlists: Customize your own playlists and manage them.
Admin Panel: Log in to the admin panel to manage users and songs.
